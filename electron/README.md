# Claw Messenger - 桌面客户端

## 🎉 有界面了！

终于有图形界面了！不用再敲命令行了！

---

## 🚀 快速开始

### 1. 安装依赖
```bash
cd electron
npm install
```

### 2. 启动应用
```bash
npm start
```

### 3. 打包成exe（可选）
```bash
npm run build-win
```

---

## 📖 使用说明

### 第一步：配置
1. 打开应用，点击"配置"标签
2. 填写飞书App ID、App Secret、Chat ID
3. 点击"保存配置"

### 第二步：发消息
1. 点击"发消息"标签
2. 输入消息内容
3. 点击"发送消息"
4. 去飞书群里看消息！

### 第三步：查看状态
1. 点击"状态"标签
2. 查看各个渠道的状态
