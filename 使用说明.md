# Claw Messenger - 使用说明

## 🚀 快速开始（Windows用户）

### 方法一：双击启动（推荐）
1. 双击 `start.bat`
2. 跟着提示走就可以了！

### 方法二：命令行启动
```bash
# 1. 安装依赖
npm install

# 2. 复制配置文件
copy .env.example .env

# 3. 编辑 .env 文件，填写飞书配置

# 4. 运行测试
node quick-test.js
```

---

## 📝 配置飞书

### 第一步：获取飞书应用凭证
1. 打开飞书开放平台：https://open.feishu.cn/
2. 创建企业自建应用
3. 获取 `App ID` 和 `App Secret`

### 第二步：获取Chat ID
1. 在飞书中创建一个群聊
2. 添加机器人到群聊
3. 获取群聊的 `Chat ID`

### 第三步：填写配置
打开 `.env` 文件，填写：
```env
FEISHU_APP_ID=你的App ID
FEISHU_APP_SECRET=你的App Secret
FEISHU_CHAT_ID=你的Chat ID
```

---

## 🎯 测试

运行 `start.bat` 或 `node quick-test.js`，如果一切正常，你会在飞书群里收到测试消息！

---

## 📦 打包成exe（可选）

如果你想要exe文件，可以用pkg打包：

```bash
# 1. 安装pkg
npm install -g pkg

# 2. 打包
pkg quick-test.js --output claw-messenger.exe
```

---

## ❓ 常见问题

### Q: 提示"请先配置 .env 文件"
A: 复制 `.env.example` 为 `.env`，然后填写飞书配置。

### Q: 飞书消息发不出去
A: 检查：
1. App ID和App Secret是否正确
2. Chat ID是否正确
3. 机器人是否已加入群聊
